  • Login brings you to dashboard. Need to track the user information through out the session until the user log off.
  • Dashboard. Show all question, number of answer each question has, action to show the current answers or add answer.
  • New question, should be 10 charcter long. Description is optional. After submitting a question it redirects to dashboard where the new question shows. Show in the top or bottom the project did not specify, for better user experince I moved it to top.
  • Cancel button should take you back to home page.
  • From dashboard you can answer to each question. Question page should show the question and its description. The answer needs to be at least 5 charcter long. Submit takes you back to dashboard with increasign the answer count for the question.
  • Q and A page can be accessed from dashboard show button. It should show the question and all the answers it recived. SHow the person name who answerd it. It have like option you, you cna click on like as much as you want to. The answers sort by likes with the most like item on top.
  • There is a link from Q and A page to Answer page and from Answer page to Q and A page. Need to keep track of the question information during the back and forth.
